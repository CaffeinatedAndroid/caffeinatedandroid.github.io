What is the Atlas Mod?

This project is where I apply new knowledge in order to develop a complete robotics platform using the generation 1 Robis as a chasis. This will allow me to expand into building completely new robots and hardware. This also serves as a platform for me to try new AI architectures and research robot embodiment.

Initial Prototyping

I started by creating an upgrade for the eyes that use round LCDs to allow further expression, these tie into a SBC, initially I started with a Raspberry Pi Zero 2w however the limited ram created issues. The current SBC is the Orangepi Zero 2w 4G, optimally I'd like to use a 8GB or 16GB Radxa Zero but for the sake of development cost, the Orangepi will do.
